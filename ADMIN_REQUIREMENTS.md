# Admin Panel Requirements

## Pages Structure:

### 1. Overview (Dashboard)
- Analytics cards
- Top products
- Quick actions

### 2. Users
- Table: ID, Email, Phone, Type, Status
- Click user → Full profile + all orders

### 3. Transactions (Orders)
- Table: Order ID, Customer, Type, Amount, Status
- Click order → Full details, products, VAT, packing slip

### 4. Products
- Table of all products
- Add/Edit/Delete products
- Categories management

### 5. Admin Users  
- Table of admin users
- Add new admin

### 6. Terms & Conditions
- Editor for T&C content

## Features Needed:
- Sidebar navigation
- Individual detail pages
- CRUD operations
- PDF/Print functionality
